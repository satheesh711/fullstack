1.what is css?why do we use it?

CSS (Cascading Style Sheets) is used to style and layout web pages — for example, to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features

2.In how many ways can we create css styling?What are they? 
            There are three primary ways to style using CSS:

            Inline CSS: Styles applied directly to HTML elements using the style attribute.
            Internal CSS: Styles defined within a "style" tag in the HTML document's "head" section.
            External CSS: Styles written in a separate .css file and linked to the HTML document.
3.d.f b/w 3 ways of css styling? 
            Inline CSS: Styles applied directly to elements; quick but not reusable.
            Internal CSS: Styles in "style" tag in HTML; good for single-page styling.
            External CSS: Separate .css file; best for reusable and maintainable styles across multiple pages.
4.What is specificity among the ways of css styling?

Specificity Order: Inline > Internal > External. Higher specificity overrides lower.

5.What is the syntax of css?

The syntax of CSS consists of a selector followed by a declaration block:

Selector: Targets HTML elements (e.g., p for paragraph).

Declaration Block: Contains property-value pairs (e.g., color: blue;).

6.What is a selector?

A selector in CSS is a pattern used to select and apply styles to HTML elements. It defines which elements on a webpage will be affected by the CSS rules.

7.How many types of selectors? 
            There are several types of CSS selectors, which can be categorized as follows:

            1. Basic Selectors:
                Element Selector: Targets HTML elements (e.g., div, p).
                Class Selector: Targets elements with a specific class (e.g., .classname).
                ID Selector: Targets a specific element with an ID (e.g., #idname).
            2. Group Selectors:
                Combines multiple selectors to apply the same styles (e.g., h1, h2, h3 { color: blue; }).
            3. Attribute Selectors:
                Selects elements based on attributes (e.g., [type="text"]).
            4. Pseudo-class Selectors:
                Targets elements based on their state (e.g., :hover, :focus).
            5. Pseudo-element Selectors:
                Targets a specific part of an element (e.g., ::before, ::first-line).
            6. Combinators:
                Descendant Selector: Selects elements that are descendants of a specified element (e.g., div p).
                Child Selector: Selects elements that are direct children of a specified element (e.g., ul > li).
                Adjacent Sibling Selector: Selects an element that is immediately next to another (e.g., h1 + p).
                General Sibling Selector: Selects all siblings of a specified element (e.g., h1 ~ p).
8.What are the general selectors? (id,class,*, (a1,a2),h2) 
            General Selectors:
                Universal: * selects all elements.
                Type: Selects specific elements (e.g., p).
                Class: .classname selects by class.
                ID: #idname selects by ID.
                Attribute: [attr="value"] selects by attribute.
                Descendant: ancestor descendant selects nested elements.
                Child: parent > child selects direct children.
                Adjacent Sibling: first + second selects the next sibling.
                General Sibling: first ~ siblings selects all siblings.
9.What is d/f b/w the pseudo class vs pseudo element? 
            Pseudo-class: Targets elements based on their state or position (e.g., :hover, :nth-child).
            Pseudo-element: Styles a specific part of an element (e.g., ::before, ::first-line).
10.What is the rule set?

A rule set in CSS consists of a selector and a declaration block that defines the styles to be applied to the selected elements.

11.What is box-modal? 
            The box model in CSS describes the layout of elements, consisting of four components:

                Content: The actual content of the element.
                Padding: Space between the content and the border.
                Border: The line surrounding the padding and content.
                Margin: The outer space separating the element from others.
12.What are the box-modal properties? (explain)

The box model properties include width, height, padding, border, and margin, defining the dimensions and spacing of elements.

13.What is the d/f b/w padding and margin?

Padding is the space between the content and the border, while margin is the space outside the border that separates the element from others.

14.What is d/f b/w outline and border?

Outline is a line drawn around an element outside its border that does not affect layout, while border is a line that wraps around the padding and content, affecting the element's size.

15.types of colors we have in css? (direct clr,rgb,rgba,hexa,,hsl,hsla)

CSS supports colors in various formats: named colors, hexadecimal, RGB, RGBA, HSL, and HSLA.

16.What is opacity?

Opacity is a CSS property that determines the transparency level of an element, ranging from 0 (completely transparent) to 1 (completely opaque).

17.What are shadows in css and how many are they?

In CSS, shadows include **box shadow**, which adds depth around an element, and **text shadow**, which adds depth to text, both customizable with offset, blur, and color.

18.What are gradients in css and how many are they? 
            In CSS, gradients are smooth transitions between two or more colors, and there are three main types:

            Linear Gradient: Transitions colors along a straight line.

                Syntax: background: linear-gradient(direction, color-stop1, color-stop2);
            Radial Gradient: Transitions colors outward from a central point.

                Syntax: background: radial-gradient(shape size at position, color-stop1, color-stop2);
            Conic Gradient: Transitions colors around a central point in a circular manner.

                Syntax: background: conic-gradient(from angle, color-stop1, color-stop2);
19.What is the d/f b/w display and float? 
            display - It controls how an element is displayed and its behavior in relation to other elements. It defines whether the element behaves as a block, inline, flex, grid, or other layout model.
            float - It positions an element to the left or right and allows text or other inline elements to wrap around it.
20.What is d/f b/w the display:inline vs display:block vs display:inline-block?

display: inline allows elements to flow without starting a new line, display: block forces elements to start on a new line and take the full width, while display: inline-block combines both, allowing elements to sit next to each other while still accepting width and height properties.

21.what is d/f b/w the visibility:hidden and display:none? 
            visibility: hidden: The element is hidden but still takes up space in the layout. Other elements won't move.
            display: none: The element is hidden and removed entirely from the document flow, meaning its space is no longer occupied and the surrounding elements adjust accordingly.
22.what is object-fit ? and in how many types can we fit the object? 
            The CSS object-fit property is used to specify how an img or video should be resized to fit its container.
            fill - This is default. The image is resized to fill the given dimension. If necessary, the image will be stretched or squished to fit
            contain - The image keeps its aspect ratio, but is resized to fit within the given dimension
            cover - The image keeps its aspect ratio and fills the given dimension. The image will be clipped to fit
            none - The image is not resized
            scale-down - the image is scaled down to the smallest version of none or contain
